About

A Streamlit-based dashboard built for Amundata to help Sudanese farmers monitor crop health using satellite remote sensing data. The platform retrieves vegetation indices from Sentinel Hub and visualizes them on interactive maps, enabling data-driven agricultural decisions.

How It Works

  1. 1 Farmers' field boundaries are stored as GeoJSON polygons with crop type metadata
  2. 2 The app queries Sentinel Hub API for satellite data over each field
  3. 3 Vegetation indices (LAI, CAB, FCOVER) are computed from multispectral bands
  4. 4 Results are visualized as colored maps and time series charts
  5. 5 Farmers identify problem areas and track crop health over the season
